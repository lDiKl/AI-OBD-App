# AI Assistant for Auto Services (B2B SaaS)

## 1. Product Vision

Web-based SaaS platform for auto repair shops that: - Analyzes OBD
logs - Suggests diagnostic paths - Generates client-ready reports -
Helps create repair estimates

Goal: Increase service efficiency and professionalism.

------------------------------------------------------------------------

## 2. Target Audience

-   Independent auto repair shops
-   Small service networks
-   Diagnostic specialists
-   Fleet maintenance companies

------------------------------------------------------------------------

## 3. Core Problem

Mechanics: - Spend time researching rare faults - Manually prepare
client explanations - Manually prepare repair estimates - Lack
structured digital knowledge base

------------------------------------------------------------------------

## 4. MVP Scope (Version 1.0)

### 4.1 OBD Log Analysis

Input: - Error codes - Freeze frame data - Optional symptoms

Output: - Likely causes (ranked) - Suggested diagnostic steps -
Recommended inspection checklist

### 4.2 Client Report Generator

-   Plain-language explanation
-   Risk summary
-   Recommended repair
-   Export to PDF

### 4.3 Repair Estimate Helper

-   Manual selection of parts
-   Labor hour suggestion
-   Editable cost calculation

### 4.4 Case History Storage

-   Save diagnostic cases
-   Search past issues
-   Build internal knowledge base

------------------------------------------------------------------------

## 5. Not Included in MVP

-   Direct OBD hardware integration
-   Full parts supplier integration
-   Automated VIN decoding
-   AI learning from aggregated industry data

------------------------------------------------------------------------

## 6. Monetization

Subscription SaaS: - Basic Plan: 49€/month - Pro Plan: 99€/month -
Multi-user plan later

------------------------------------------------------------------------

## 7. Technical Architecture

Frontend: - React / Vue web application

Backend: - REST API - Secure authentication - AI engine (LLM +
structured knowledge base) - PDF generation service

------------------------------------------------------------------------

## 8. Estimated Development Time

-   Backend logic: 4--5 weeks
-   Frontend dashboard: 3--4 weeks
-   PDF/report system: 1--2 weeks
-   Testing & onboarding flow: 2 weeks

Estimated MVP timeline: \~3 months (solo developer).
