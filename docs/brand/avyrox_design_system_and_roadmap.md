# 5. Android Icon System

App icons should be simple and recognizable.

Recommended concept:

Main symbol:

Stylized **A** shaped like a road or signal.

Alternative concepts:

1.  Signal waves from a car
2.  Car + cloud
3.  Diagnostic sensor icon

------------------------------------------------------------------------

Icon colors

Background:

Primary blue (#1E3A8A)

Symbol:

White

Accent:

Cyan (#22D3EE)

Launch the Avyrox ecosystem with two apps and a backend platform.

------------------------------------------------------------------------

Tasks:

-   finalize brand
-   register domain
-   register trademark
-   product specification
-   technical architecture

------------------------------------------------------------------------

Month 3--4

MVP backend

Build:

-   user accounts
-   vehicle database
-   diagnostics API
-   service directory

Technology examples:

Backend:

Python / Node.js

Database:

PostgreSQL

Month 5--6

Driver App (Avyrox Drive)

Features:

-   OBD connection
-   error scanning
-   error explanations
-   vehicle dashboard

Goal:

internal alpha version

------------------------------------------------------------------------

Month 7--8

-   service profiles
-   repair estimates
-   job tracking
-   messaging
